## ⏱️ 快速解决（只需 7 分钟）

### 步骤 1：创建飞书应用（2 分钟）

打开 https://open.feishu.cn

1. 点击 "我的应用"
2. 点击 "创建应用"
3. 填写应用名称：`小红书自动发布`
4. 点击 "创建"

### 步骤 2：获取凭证（1 分钟）

在应用详情页找到 "凭证与基础信息"：
- 复制 **App ID** ← 保存这个
- 复制 **App Secret** ← 保存这个

### 步骤 3：添加权限（1 分钟）

1. 左侧菜单 → "权限管理"
2. 点击 "添加权限"
3. 搜索 `bitable:app:readonly` → 添加
4. 搜索 `bitable:table:readonly` → 添加
5. 点击 "保存"

### 步骤 4：发布应用（1 分钟）

点击 "发布" 按钮

**⚠️ 重要：必须发布！**

### 步骤 5：配置插件（1 分钟）

打开插件 → "配置" 标签：

1. **飞书 App ID**: 粘贴你的 App ID
2. **飞书 App Secret**: 粘贴你的 App Secret
3. **飞书表格 ID**: `GGh2bW3Q2aHpi1shiVqcAlhmnMd`
4. 点击 "保存配置"

### 步骤 6：测试连接（1 分钟）

点击 "测试连接" 按钮

**预期结果：**
```
✅ 连接成功！找到 X 个表格，Y 条记录
```

### 步骤 7：开始使用（1 分钟）

1. 在飞书表格中添加待发布内容
2. 设置状态为 "pending" 或 "待发布"
3. 在插件中点击 "发布" 标签
4. 点击 "开始发布"
5. 在 "日志" 标签中查看进度

## ❌ 如果测试连接失败

### 错误代码 1
**原因**: App ID 或 Secret 错误
**解决**: 重新复制凭证，确保没有空格

### 错误代码 6
**原因**: 应用未发布
**解决**: 在飞书开放平台点击 "发布" 按钮

### 错误代码 91402
**原因**: Base ID 错误
**解决**: 检查 Base ID 是否为 20+ 个字母和数字

### 错误代码 99991001
**原因**: 权限不足
**解决**: 添加 `bitable:app:readonly` 权限并重新发布
