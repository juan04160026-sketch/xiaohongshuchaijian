# 快速开始 - 5 分钟设置

## 你需要做的事

### 1️⃣ 获取飞书凭证（2 分钟）

打开 https://open.feishu.cn

**如果没有应用：**
- 点击 "我的应用" → "创建应用"
- 填写应用名称，点击创建

**获取凭证：**
- 在应用详情页找到 "凭证与基础信息"
- 复制 **App ID**
- 复制 **App Secret**

### 2️⃣ 添加权限（1 分钟）

在应用详情页：
- 左侧菜单 → "权限管理"
- 点击 "添加权限"
- 搜索并添加：
  - `bitable:app:readonly`
  - `bitable:table:readonly`
- 点击 "保存"

### 3️⃣ 发布应用（1 分钟）

在应用详情页：
- 点击 "发布" 按钮
- **重要：必须发布！**

### 4️⃣ 在插件中配置（1 分钟）

打开插件 → "配置" 标签：
- **App ID**: 粘贴你的 App ID
- **App Secret**: 粘贴你的 App Secret
- **Base ID**: `GGh2bW3Q2aHpi1shiVqcAlhmnMd`
- 点击 "保存配置"
- 点击 "测试连接"

### 5️⃣ 开始发布（1 分钟）

- 在飞书表格中添加待发布内容
- 设置状态为 "pending" 或 "待发布"
- 在插件中点击 "发布" 标签
- 点击 "开始发布"

---

## 如果测试连接失败

### 错误代码 1
❌ App ID 或 Secret 错误
✅ 重新复制，确保没有空格

### 错误代码 6
❌ 应用未发布
✅ 在飞书开放平台点击 "发布"

### 错误代码 91402
❌ Base ID 错误
✅ 检查 Base ID 是否为 20+ 个字母和数字

### 错误代码 99991001
❌ 权限不足
✅ 添加 `bitable:app:readonly` 权限
