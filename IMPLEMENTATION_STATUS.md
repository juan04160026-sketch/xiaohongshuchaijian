## 🔧 解决方案

### 需要做的事（7 分钟）

1. **创建飞书应用** (2 分钟)
   - 打开 https://open.feishu.cn
   - 创建应用
   - 获取 App ID 和 App Secret

2. **添加权限** (1 分钟)
   - 添加 `bitable:app:readonly`
   - 添加 `bitable:table:readonly`

3. **发布应用** (1 分钟)
   - 点击 "发布" 按钮

4. **配置插件** (1 分钟)
   - 填写 App ID、App Secret、Base ID
   - 点击 "保存配置"

5. **测试连接** (1 分钟)
   - 点击 "测试连接"
   - 验证成功

6. **准备数据** (1 分钟)
   - 在飞书表格中添加待发布内容
   - 设置状态为 "pending"

7. **开始发布** (1 分钟)
   - 点击 "开始发布"
   - 监控日志

## 💬 常见问题

**Q: 为什么连接不通？**
A: 需要配置飞书 API 凭证。按照 `QUICK_START_SETUP.md` 的步骤操作。

**Q: 如何获取 App ID 和 App Secret？**
A: 在飞书开放平台创建应用后，在应用详情页找到 "凭证与基础信息"。

**Q: 应用创建后为什么还是连接失败？**
A: 最常见的原因是应用未发布。点击 "发布" 按钮发布应用。

**Q: Base ID 在哪里获取？**
A: 打开你的飞书多维表格，从 URL 中复制 Base ID。
