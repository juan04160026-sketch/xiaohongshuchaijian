# 飞书连接问题 - 快速解决指南

## 🎯 你的情况

- ✅ 插件已完全开发好
- ✅ 代码已准备好读取飞书数据
- ❌ 飞书连接不通（需要配置凭证）

---

## 🚀 快速解决（7 分钟）

### 1️⃣ 打开飞书开放平台
https://open.feishu.cn

### 2️⃣ 创建应用
- 点击 "我的应用" → "创建应用"
- 填写应用名称：`小红书自动发布`
- 点击 "创建"

### 3️⃣ 获取凭证
在应用详情页找到 "凭证与基础信息"：
- 复制 **App ID**
- 复制 **App Secret**

### 4️⃣ 添加权限
- 左侧菜单 → "权限管理"
- 添加权限：`bitable:app:readonly` 和 `bitable:table:readonly`
- 点击 "保存"

### 5️⃣ 发布应用
- 点击 "发布" 按钮
- **重要：必须发布！**

### 6️⃣ 在插件中配置
打开插件 → "配置" 标签：
- App ID: 粘贴你的 App ID
- App Secret: 粘贴你的 App Secret
- Base ID: `GGh2bW3Q2aHpi1shiVqcAlhmnMd`
- 点击 "保存配置"
- 点击 "测试连接"

### 7️⃣ 验证成功
看到 `✅ 连接成功！` 就完成了！

---

## ❌ 如果测试失败

### 错误代码 1
❌ App ID 或 Secret 错误
✅ 重新复制，确保没有空格

### 错误代码 6
❌ 应用未发布
✅ 在飞书开放平台点击 "发布"

### 错误代码 91402
❌ Base ID 错误
✅ 检查 Base ID 是否为 20+ 个字母和数字

### 错误代码 99991001
❌ 权限不足
✅ 添加 `bitable:app:readonly` 权限并重新发布
