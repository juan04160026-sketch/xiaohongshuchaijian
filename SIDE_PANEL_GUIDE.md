# 侧边栏面板使用指南

## 📌 什么是侧边栏面板？

侧边栏面板是一个固定在浏览器右侧的面板，可以在浏览任何网页时保持可见。不像弹窗会在切换页面时消失，侧边栏面板会一直显示。

## 🚀 如何打开侧边栏

### 方式 1: 点击扩展图标

1. 在 Chrome 工具栏找到"小红书自动发布助手"图标
2. 点击图标
3. 侧边栏会在浏览器右侧打开

### 方式 2: 快捷键

- **Windows/Linux**: `Ctrl + Shift + Y`
- **Mac**: `Cmd + Shift + Y`

（如果快捷键不工作，可以在 Chrome 设置中自定义）

## ✨ 侧边栏的优势

### 1. 持久显示
- 切换页面时不会消失
- 可以在任何网页上使用
- 不占用主窗口空间

### 2. 实时监控
- 在浏览小红书时实时查看发布进度
- 随时查看日志信息
- 及时发现问题

### 3. 便捷操作
- 无需频繁打开/关闭
- 可以边浏览边监控
- 快速访问所有功能

### 4. 灵活调整
- 可以拖动侧边栏边界调整宽度
- 可以最小化侧边栏
- 可以随时关闭

## 🎯 使用场景

### 场景 1: 配置阶段
1. 打开侧边栏
2. 在"配置"标签中填写飞书凭证
3. 保存配置

### 场景 2: 发布阶段
1. 打开侧边栏
2. 切换到"发布"标签
3. 点击"开始发布"
4. 在侧边栏中实时查看进度

### 场景 3: 监控阶段
1. 打开侧边栏
2. 切换到"日志"标签
3. 实时查看发布日志
4. 如有错误立即处理

## 🔧 侧边栏功能

### 配置标签
- 输入飞书 App ID
- 输入飞书 App Secret
- 输入飞书 Table ID
- 保存配置

### 发布标签
- 显示统计信息（待发布、已发布、失败）
- 显示发布进度条
- 开始/暂停/停止按钮
- 实时状态提示

### 日志标签
- 显示最新 50 条日志
- 搜索日志功能
- 清空日志按钮
- 彩色日志显示

## 💡 最佳实践

### 1. 保持侧边栏打开
- 发布过程中保持侧边栏打开
- 便于实时监控进度
- 及时发现和处理问题

### 2. 定期查看日志
- 每次发布后查看日志
- 记录发布统计信息
- 分析失败原因

### 3. 调整侧边栏宽度
- 根据需要调整宽度
- 便于查看完整的日志信息
- 不影响网页浏览

### 4. 使用快捷键
- 记住快捷键 (Ctrl+Shift+Y)
- 快速打开/关闭侧边栏
- 提高工作效率

## 🎨 侧边栏外观

侧边栏采用现代化设计：
- 蓝色主题色
- 清晰的标签导航
- 彩色的日志显示
- 响应式布局

## 📊 侧边栏宽度

- **默认宽度**: 约 350-400px
- **最小宽度**: 约 250px
- **最大宽度**: 约 600px

可以通过拖动侧边栏左边界来调整宽度。

## 🔄 侧边栏状态保存

侧边栏会记住：
- 上次选择的标签（配置/发布/日志）
- 日志历史记录
- 配置信息

下次打开时会恢复到上次的状态。

## ⚙️ 侧边栏设置

### 打开侧边栏
```
点击扩展图标 → 侧边栏打开
```

### 关闭侧边栏
```
点击侧边栏右上角的关闭按钮
或按 Escape 键
```

### 调整宽度
```
拖动侧边栏左边界
```

### 最小化侧边栏
```
点击侧边栏顶部的最小化按钮
```

## 🐛 常见问题

### Q: 侧边栏打不开
**A**: 
1. 确保扩展已加载
2. 尝试点击扩展图标
3. 尝试使用快捷键 (Ctrl+Shift+Y)
4. 重新加载扩展

### Q: 侧边栏显示不完整
**A**:
1. 调整侧边栏宽度
2. 刷新页面
3. 检查浏览器缩放比例

### Q: 侧边栏中的日志显示不全
**A**:
1. 调整侧边栏宽度
2. 使用搜索功能过滤日志
3. 清空旧日志

### Q: 侧边栏在某些网站上不显示
**A**:
1. 侧边栏在所有网站上都应该可用
2. 尝试刷新页面
3. 检查扩展权限

## 📞 获取帮助

如果侧边栏有问题：
1. 查看浏览器控制台的错误信息
2. 查看 `TROUBLESHOOTING.md`
3. 重新加载扩展
4. 清除浏览器缓存
